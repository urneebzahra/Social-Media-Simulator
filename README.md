# Social-Media-Simulator
C++ Social Media Simulator: A final year project simulating a mini social network. Users can create profiles, post updates, like/comment on posts, and follow others. Demonstrates social media logic and basic C++ programming concepts.
This is a Social Media Simulator developed as a Final Year Project using C++. It simulates a mini social network where users can create profiles, post updates, like and comment on posts, and follow other users.

Features:
	•	User profile creation
	•	Posting, liking, and commenting
	•	Follow/unfollow functionality
	•	Simulates a small-scale social media environment

How to Run:
	1.	Clone the repository
	2.	Open the project in a C++ IDE (e.g., Code::Blocks, Visual Studio)
	3.	Compile and run the main.cpp file
